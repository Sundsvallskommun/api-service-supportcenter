# API Changelog 2.1 vs. 2.2

## POST /{municipalityId}/cases

- added the new optional request property 'imeiNumber'
- added the new optional request property 'modelName'

## GET /{municipalityId}/cases/{caseId}

- added the optional property 'imeiNumber' to the response with the '200' status
- added the optional property 'modelName' to the response with the '200' status

## PATCH /{municipalityId}/cases/{caseId}

- added the new optional request property 'imeiNumber'
- added the new optional request property 'modelName'
